8.3.4.8.13   LiteralRational

Description

A LiteralRational        is a LiteralExpression         that provides a  Rational     value as a result. Its result
parameter     must have the type   Rational.

General Classes

LiteralExpression

Attributes

value : Real

The Real value that is the result of evaluating this Expression.

The value whose rational approximation is the result of evaluating this        LiteralRational.

Operations

None.

Constraints

checkLiteralRationalSpecialization

A LiteralRational        must directly or indirectly specialize    Performances::literalRationalEvaluations
from the Kernel Semantic Library.
