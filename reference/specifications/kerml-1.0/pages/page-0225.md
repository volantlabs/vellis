8.3.4.7.5 Invariant

Description

An Invariant     is aBooleanExpression       that is asserted to have a specificBoolean   result value. IfisNegated
= false, then the result is asserted to be true. IfisNegated = true, then the result is asserted to be false.

General Classes

BooleanExpression

Attributes

isNegated : Boolean

Whether this Invariant     is asserted to be false rather than true.
