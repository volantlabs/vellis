8.3.4.4.3  AssociationStructure

Description

An AssociationStructure          is an Association     that is also a Structure, classifying link objects that are both
links and objects. As objects, link objects can be created and destroyed, and their non-end      Features    can change
over time. However, the values of the end    Features    of a link object are fixed and cannot change over its lifetime.
