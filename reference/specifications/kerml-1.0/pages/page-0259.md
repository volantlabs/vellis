8.3.4.13.2   ElementFilterMembership

Description

ElementFilterMembership             is a Membership      between a   Namespace      and a model-level evaluable
Boolean-valued      Expression, asserting that imported         members     of the Namespace      should be filtered using the
condition     Expression. A general        Namespace      does not define any specific filtering behavior, but such behavior
may be defined for various specialized kinds of        Namespaces.
