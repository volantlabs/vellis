8.3.4.5.3  Connector

Description

A Connector      is a usage of Associations, with links restricted according to instances of the          Type   in which they
are used (domain of the    Connector). The      associations       of the Connector     restrict what kinds of things might be
linked. The  Connector     further restricts these links to be between values of     Features     on instances of its domain.
