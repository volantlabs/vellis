A.7   States

The states of aVehicle    enable selected actions to be performed. The  Vehicle    exhibits itsstatevehicleStates.
This state is a parallel state, so its substatesoperatingStates   and  healthStates     are concurrent. The states
operatingStates      and healthStates      are not parallel, so only one of each of their substates can be active at any
given time.
