8.4.19.1  Analysis Case Definitions

An AnalysisCaseDefinition           is a kind ofCaseDefinition. As such, all the general semantic constraints for a
CaseDefinition      (see 8.3.22.2 ) apply to an AnalysisCaseDefinition. In addition, the
checkAnalysisCaseDefinitionSpecialization                  constraint requires that an AnalysisCaseDefinition
specialize the base AnalysisCaseDefinition          AnalysisCases::AnalysisCase            (see 9.2.16.2.1 ), which
subclassifies theCaseDefinition       Cases::Case      (see9.2.15.2.1  ).
