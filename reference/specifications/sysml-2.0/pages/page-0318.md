validateOccurrenceUsagePortionKind

If anOccurrenceUsage       has a non-null  portionKind, then its    owningType     must be an
OccurrenceDefinition         or an OccurrenceUsage.

portionKind <> null implies
     owningType <> null and
     (owningType.oclIsKindOf(OccurrenceDefinition) or
       owningType.oclIsKindOf(OccurrenceUsage))

8.3.9.5 PortionKind

Description

PortionKind     is an enumeration of the specific kinds of  Occurrence     portions that can be represented by an
OccurrenceUsage.

General Classes

None.

Literal Values

snapshot

A snapshot of an  Occurrence     (a time slice with zero duration).

timeslice

A time slice of an Occurrence     (a portion over time).
