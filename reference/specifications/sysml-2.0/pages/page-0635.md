9.8.3.2.35   UnitPrefix

Element

AttributeDefinition

Description

UnitPrefix      is an AttributeDefinition           that represents a named multiple or sub-multiple measurement unit
prefix as defined in ISO/IEC 80000-1.

General Types

None.

Features

conversionFactor : Real

A Real   that specifies the value of multiple or sub-multiple of this      UnitPrefix.

longName : String

symbol : String

The short symbolic name of this      UnitPrefix.

Examples are: "k" for "kilo", "m" for "milli", "MeBi" for "mega binary".

Constraints

None.
