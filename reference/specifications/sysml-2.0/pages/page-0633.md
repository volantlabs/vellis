9.8.3.2.32   TranslationRotationSequence

Element

AttributeDefinition

Description

A TranslationRotationSequence                 is a coordinate frame transformation specified by a sequence of translations
and/or rotations.

Note:  This is a  CoordinateTransformation              that is convenient for interpretation by humans. In particular a
sequence of rotations about the principal axes of a coordinate frame is much more easy understandable than a
rotation about an arbitrary axis. Any sequence can be reduced to a single combination of a translation and a rotation
about a particular axis, but in general the original sequence cannot be retrieved as there are infinitely many
sequences representing the reduced transformation.
