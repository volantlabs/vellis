9.8.3.2.11   DefinitionalQuantityValue

Element

AttributeDefinition

Description

A DefinitionalQuantityValue                is a representation of a particular quantity value that is used in the definition of
a MeasurementReference. Typically such a particular value is defined by convention. It can be used to define a
selected reference value, such as the meaning of zero on a measurement scale or the origin of a top-level coordinate
system.

General Types

None.

Features

definition : String

num : Number [1..*]

Constraints

None.
